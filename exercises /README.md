# Thư mục đề bài

Thư mục này chứa các đề bài / bài tập cho project.

## Nội dung

| File | Nội dung |
|------|----------|
| **01_API_Flask.md** | Bài tập viết API với Flask (JSON, GET/POST/PUT/DELETE) dựa trên model `User` có sẵn trong project. |
| **01_Danh_gia_ket_qua.md** | Đánh giá kết quả bài tập 1 dựa trên đề bài và evidences trong `evidences/01`. |
| **02_API_Authentication.md** | Bài tập xác thực API: đăng nhập (POST /api/login), token, bảo vệ route (GET /api/me, PUT/DELETE), logout. |

- **Bài 1:** Làm lần lượt từ Phần 1 đến Phần 3 trong **01_API_Flask.md**. Phần 4 dùng để tự test API (Postman, curl hoặc trình duyệt). Evidence lưu trong **evidences/01**.
- **Bài 2:** Làm theo **02_API_Authentication.md**. Evidence (chụp màn hình test API) lưu trong **evidences/02**.
